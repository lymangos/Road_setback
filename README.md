# AutoCAD 自动建筑退线生成器

读取城市规划道路网 DXF 文件，为每个闭合地块自动生成建筑退线（建筑红线）。

## 核心算法

1. **读取 DXF** — 使用 ezdxf 提取所有闭合多段线（含弧线离散化）
2. **射线检测道路宽度** — 从每条边中点沿外法线发射射线，首个交点距离即道路宽度
3. **退距映射** — 路宽 ≥35m → 退15m（主干道），≥20m → 退10m（次干道），其余 → 退5m（支路）
4. **逐边偏移** — 每条边按各自退距向内偏移，相邻偏移线求交得新顶点
5. **安全裁剪** — shapely 修复自交并与原始地块取交集，保证退线不超出地块

## 使用方法

```bash
pip install ezdxf shapely numpy
python auto_setback.py
```

- 输入：`cadfile.dxf`（当前目录下的道路网 DXF 文件）
- 输出：`output_<时间戳>.dxf`（含原始线 + 建筑红线图层 + 标注）

## 输出图层

| 图层 | 颜色 | 内容 |
|------|------|------|
| 建筑红线 | 红色 | 退线闭合多段线 |
| 标注_参考 | 黄色 | 退距与路宽标注文字 |

## 退距规则

| 道路宽度 | 道路类型 | 退距 |
|----------|---------|------|
| ≥ 35m | 主干道 | 15m |
| ≥ 20m | 次干道 | 10m |
| < 20m | 支路 | 5m |
| < 3m | 紧邻地块 | 5m |
| 无交点 | 边界 | 5m |
